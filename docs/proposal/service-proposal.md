## サービス概要

FaultRay は、貴社のITインフラ構成をもとに **100〜400の障害シナリオを自動生成・数学的にシミュレーション**（構成規模に応じて最大2,000）し、システムが本当に止まらないかをレポートするサービスです。本番環境に一切手を加えず、ヒアリングから最短数日でレポートを納品します。

---

## なぜ今、障害シミュレーション診断が必要か

### 1. IPA「情報セキュリティ10大脅威 2026」でインフラ関連リスクが上位

IPAが毎年発表する10大脅威において、ITインフラへの攻撃・障害は組織部門で継続的に上位にランクインしています。「起きてから対応する」では事業継続が間に合いません。

### 2. 個人情報保護法・安全管理措置義務の強化

改正個人情報保護法により、個人データを扱うすべての事業者は「技術的安全管理措置」の実施が義務付けられています。障害時のデータ保護体制を**事前に証明できるか**が問われています。

### 3. CrowdStrike障害（2024年）の教訓 — 推定被害額54億ドル

2024年7月のCrowdStrike社ソフトウェア更新に起因する世界規模のシステム障害は、航空・金融・医療など幅広い業種に影響を与え、推定54億ドルの被害をもたらしました。**「信頼しているベンダーが起因でも自社が止まる」**という現実をこの事故は示しています。事前のシミュレーションで「どこが連鎖障害の引き金になるか」を把握しておくことが、被害を最小化する唯一の方法です。

## サービスの流れ

| ステップ | 内容 | 所要時間 |
|---|---|---|
| **① ヒアリング** | 現在のインフラ構成（サーバー・DB・CDN等）を30分でヒアリング | 30分 |
| **② YAML変換** | ヒアリング内容をFaultRay専用の構成定義ファイル（YAML）に変換 | 1〜2営業日 |
| **③ シミュレーション実行** | 134〜2,000以上の障害シナリオを自動生成・実行 | 即時（ツール内） |
| **④ レポート納品** | 経営層向けエグゼクティブレポート + 詳細技術レポートをHTMLで納品 | 最短翌営業日 |

## 分析機能一覧

| 機能 | 内容 |
|---|---|
| **障害シミュレーション** | 100〜400のシナリオを自動生成（単一障害・複合障害・トラフィックスパイク等。大規模構成で最大2,000） |
| **カスケード障害分析** | あるコンポーネントの障害が連鎖してどこまで波及するかをトレース |
| **Monte Carlo分析** | 10,000試行による可用性（稼働率）の統計的算出 |
| **FMEA（故障モード分析）** | 全コンポーネントの故障モード・影響・リスク優先度（RPN）を評価 |
| **コンプライアンスチェック** | SOC2 / ISO27001 / DORA（EU規制）対応状況の自動評価 |
| **SLA検証** | 目標可用性（例：99.9%）が統計的に達成可能かを証明 |
| **エグゼクティブレポート** | 経営層・役員向けのサマリーHTMLレポートを自動生成 |
| **リスク財務試算** | 障害発生時の推定損失額・ビジネスインパクトを算出 |
